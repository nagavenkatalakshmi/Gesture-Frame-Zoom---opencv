GestureZoom - Picture Magnification through Hand Gestures with OpenCV

GestureZoom is an innovative application that utilizes computer vision and hand gesture recognition to enable Frame magnification. Built with OpenCV, this project empowers users to control the zoom level of an Frame by performing specific hand gestures in front of a webcam.

  • GestureZoom provides a real-time and interactive experience, allowing users to control the zoom level dynamically.
  • The application uses the MediaPipe library to recognize hand gestures captured by the webcam
  • Users can perform intuitive hand movements to zoom in or out, adjusting the magnification factor on the frame.
  • The zooming process is responsive and smooth, enhancing the user experience.
